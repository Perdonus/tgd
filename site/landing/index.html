Why Astrogram

Not just a rebrand, but a desktop client with its own priorities.

Native plugin runtime

`.tgd` plugins load as native modules with window APIs, command interceptors, settings pages and recovery-aware logs.

Privacy controls

Ghost mode, anti-recall logging and client-side controls live directly in Astrogram instead of being hidden behind random patches.

Expressive interface

Warmer colors, stronger hierarchy, better cards and more room for desktop-first interactions than stock Telegram Desktop.

Recovery that matters

Tracked installs, plugin logs, safe mode and plugin quarantine make debugging less chaotic when native code breaks.

Plugin platform

Astrogram keeps plugin power native, but pushes fragile UI back into the client.

What ships today

  • Host-rendered settings pages with toggles, buttons, inputs and sliders.
  • Window/session hooks for plugins like AstroTransparent, AI Chat and diagnostics.
  • Safe mode, runtime docs and package metadata that stay visible by default.

Where to go next

  • Read the runtime docs and package format at docs.astrogram.su.
  • Follow Windows builds from GitHub Actions while releases are still moving fast.
  • Join the channel and chat to track new client-side features and plugin API changes.
